# magister
Project for teachers and students to commit specific / scientific work without needance of thinking about personally document exchange.

Stack:
- HTML; 
- CSS (Sass, FlexBox);
- JS (jQuery);
- PHP (RedBeanPhp).

Src now: http://zss.sut.ru/magister/
